# 调酒助手 · 你的数字吧台档案员

一款手机端调酒创作助手：记录每款酒的配方、形态照片、灵感来源与故事文案，AI 辅助命名与复盘。

## 功能

- **酒款档案库**：配方、比例、技法、装饰、风味关键词、灵感来源、成品照片、备选命名、故事文案、客人反馈，全字段归档
- **灵感速记**：做完酒两分钟快速录入，支持顺手拍一张成品照
- **命名与叙事**：AI 给出 4 个命名选项及故事线，支持文案润色（默认 Kimi/Moonshot 接口，兼容 OpenAI 格式，API Key 仅存本机）
- **创作简报**：近 7/30 天统计、创作管线、卡壳提醒、风味足迹、系列图谱、AI 复盘
- **个人档案**：助手人格（沉默助手 / 挑剔评论家 / 灵感推送员）、偏好与禁忌注入 AI 提示词

## 技术栈

React 19 + TypeScript + Vite + Tailwind CSS + shadcn/ui，数据存于浏览器 localStorage，无后端。

## 本地运行

```bash
npm install
npm run dev      # 开发预览
npm run build    # 构建到 dist/
```

## 部署

构建产物在 `dist/`，为纯静态文件（base 已设为相对路径），可直接部署到 GitHub Pages、Vercel、Netlify 或任意静态托管。
